Senior Software Engineer

Google LLC • Mountain View, CA (Hybrid)

Job Description

Google's software engineers develop the next-generation technologies that change how billions of users connect, explore, and interact with information. We are looking for a Senior Software Engineer to join our Core Cloud Platform Team. In this role, you will build and maintain highly distributed infrastructure systems that scale automatically to support our global cloud customers.

Key Responsibilities

  • Design, develop, and maintain secure, scalable distributed server systems in cloud infrastructure frameworks.
  • Write clean, tested, and highly optimized code in Go, Python, or Java.
  • Collaborate closely with product managers and database engineers to deliver new security configurations.
  • Mentor and guide junior and mid-level software engineering team developers.
  • Debug production infrastructure bugs and handle deployment pipelines on Kubernetes clusters.

Skills & Experience Required

  • Bachelor's degree in Computer Science, engineering, or equivalent professional work experience.
  • 5+ years of software development experience writing code in Go, Python, C++, or Java.
  • Strong experience in cloud server systems architecture, API designs, and container structures (Kubernetes/Docker).
  • Experience working with SQL/NoSQL databases and data pipeline clusters.
  • Excellent communication skills and leadership mindset.

What We Offer

  • Competitive base salary range ($160,000 - $210,000 /yr) + annual performance bonus + stock options.
  • Premium health, dental, and vision insurance policies.
  • 401(k) retirement matching savings plans.
  • Flexible work setups (Hybrid - 3 days in office, 2 days remote).
  • State-of-the-art office setups, on-site cafes, and wellness gyms.
